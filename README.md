# 💌 Приглашение на свидание

Романтичный одностраничный сайт-приглашение. Тёмный фон с золотыми акцентами, плавные переходы между экранами, кнопка «Нет», которая убегает.

---

## Публикация на GitHub Pages

1. Создайте новый репозиторий на GitHub (например `date-invitation`).
2. Загрузите три файла: `index.html`, `style.css`, `script.js`.
3. Откройте **Settings → Pages**.
4. В разделе **Source** выберите:
   - Branch: `main`
   - Folder: `/ (root)`
5. Нажмите **Save**.
6. Через 1–2 минуты сайт появится по адресу:

   ```
   https://<ваш-username>.github.io/date-invitation/
   ```

---

## Файлы

```
├── index.html   — структура (4 экрана)
├── style.css    — тёмная романтичная тема, адаптивная
├── script.js    — анимации, переходы, конфетти, localStorage
└── README.md    — эта инструкция
```

---

## Что внутри

| Экран | Описание |
|-------|----------|
| **1** | «Пойдёшь со мной на свидание?» — кнопка «Нет» убегает от курсора (десктоп) или телепортируется при касании (мобильные) |
| **2** | Радостное подтверждение |
| **3** | Выбор места карточками + дата и время |
| **4** | Карточка-приглашение с анимацией конфетти; данные сохраняются в `localStorage` |

---

## Локальный запуск

Достаточно открыть `index.html` в браузере — никаких зависимостей и сборки не нужно.
